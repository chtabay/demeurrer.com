## Schéma JSON (`categories.json`)

```
{
  slug            string     identifiant URL-safe
  name            string     nom affiché
  emoji           string     icône de carte
  difficulty      1 | 2 | 3  niveau de difficulté
  formula         string     expression LaTeX (rendue par KaTeX)
  correctAnswer   number     réponse numérique attendue
  inputMode       string     type de clavier ("decimal")
  smart           { name, url }
  dumb            { type: "fixed", name, url }
                  | { type: "random", pool: [{ name, url }] }
  easterEggs      [{ answer: number|string, name, url }]
  tags            string[]   mots-clés pour la recherche
  type?           "political"  (carte spéciale, pas de formule)
}
```

---

## Règles d'affichage du résultat

Le résultat ne comporte jamais de commentaire. Deux éléments uniquement :

1. **Le nom du site** (typographie stylée selon le type : vert/smart, rouge/dumb, accent/egg)
2. **Un lien** : « Accéder au site → »

Pas d'emoji de verdict. Pas de message. Pas d'encouragement.
Le site énonce. L'utilisateur en fait ce qu'il veut.
